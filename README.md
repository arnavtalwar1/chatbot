# Building AI Model

This project demonstrates the process of building an AI/Machine Learning model using Python in a Jupyter Notebook.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training
- Model evaluation
- Performance metrics

## Technologies Used
- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/building-ai-model.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

## Run

Open the notebook:

```bash
jupyter notebook Building_AI_model.ipynb
```
